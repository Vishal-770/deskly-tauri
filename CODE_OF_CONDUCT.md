# Code of Conduct

Deskly is an open-source project built for university students. We aim to foster a collaborative, productive, and respectful environment for developers and users alike.

---

## Guidelines

- **Be Respectful**: Treat all contributors, maintainers, and users with respect. Constructive disagreement and technical discussions are welcome; toxicity and personal attacks are not.
- **Constructive Feedback**: Provide helpful, actionable feedback in Pull Requests and Issues. Focus on the code and implementation details.
- **Collaborative Problem Solving**: Work together to fix bugs, improve features, and enhance security while respecting user privacy.

---

## Unacceptable Behavior

- Trolling, insulting, or derogatory comments.
- Personal attacks or targeted harassment.
- Posting private user information, log dumps containing real student credentials, or session tokens.
- Intentional spamming or abusive issue submissions.

---

## Enforcement

Maintainers reserve the right to close issues, reject pull requests, or restrict access to repository discussions for contributions that violate these standards.

To report any concerns, please open a issue on [GitHub Issues](https://github.com/Vishal-770/deskly-tauri/issues).
